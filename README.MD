# ①課題名 
Generative Life Integration Tool

## ②課題内容（どんな作品か） 
GoogleChromeの拡張機能（新タブ開いた際のスペース）×LINE（未実装）
chromeの新タブを開くたびに目標とメモとビジョンが提示
AIがどう伴走するかは今後

① メモ機能（分散メモの集約）
自分がいいなと思った言葉・考え方がLINE、Word、iPhoneメモ等が
散らばっているので1箇所に集約出来る機能

② セレンディピティ機能
埋もれてしまった過去のメモを、AIが最適なタイミングで再提示し、
気づきを与える。不要な情報は削除提案も

③ 高解像度の目標設定
単なる自由記述ではなく、AIと対話しフレームワークに基づいた「問い」に答えることで、曖昧な目標（例：経営者になりたい）を、解像度の高い自分ごとの文章へ昇華。

④ ビジョンボード
言語化された目標に対し、具体的なイメージ画像（ex理想のライフスタイル）を
貼り付け、視覚的に定着させる。
